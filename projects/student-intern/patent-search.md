## 주요 업무 및 상세 역할

### Text Search
- Elastic Search, API를 활용하여 JSON 데이터를 Front-end로 전달
- txt 파일을 따로 두어, 검색어 자동 완성 기능 구현
- 검색 결과에 따른 페이지네이션 구현
- Patent 고유 ID에 따른 URL 생성 후 Google Patent로 연동
