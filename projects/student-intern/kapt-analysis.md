## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 작업 기간 | 2022.07.04 ~ 2022.08.29 |
| 인력 구성도(기여도) | 조석영(33%), 데이터 팀 2명(66%) |
| 프로젝트 목적 | 전기차 충전기 설치 관련 법률 위반 아파트, 추가 설치가 필요한 아파트 정보 제공 |
| 프로젝트 내용 | 법률에 따라 전기차 충전기 대수를 파악하고 분석하여 해당 정보를 정부에 제공 |

## 주요 업무 및 상세 역할

- 국내 아파트의 전기차 충전기 현황 분석
- K-APT 공동주택관리정보시스템 데이터 정제 및 가공
- 국토교통부 공동주택 단지 데이터 정제 및 가공
- 두 데이터를 merge하여 정제 및 SQL 서버에 업로드 (feature 정의)
- 전기차 충전기 설치 법률에 따른 아파트 충전기 설치 현황 비교
- 충전기 설치가 더 필요한 아파트, 법률에 위반되는 방식으로 설치된 충전기 등의 정보를 제공 (Tableau로 대시보드 제작)
