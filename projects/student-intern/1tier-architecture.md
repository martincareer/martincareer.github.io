## 주요 업무 및 상세 역할

- WS와 WAS를 연동하여 웹 브라우저에서 WS의 주소로 WAS(TOMCAT) 페이지 접속
- WAS와 DB를 연동하고, WAS 페이지에 확인 연동 문구 및 SQL Query 문 실행 결과 출력
