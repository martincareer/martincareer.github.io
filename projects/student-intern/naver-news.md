## 주요 업무 및 상세 역할

- Python 크롤링을 통해 네이버 뉴스를 키워드별로 수집
- 데이터 정제 및 가공(전처리)
- Docker를 활용하여 DB Server 구축(On-Premise)
- DB Server에 뉴스 데이터 적재
- 키워드에 따라 검색할 수 있는 UI 제작
