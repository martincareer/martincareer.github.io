## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 작업 기간 | 2022.10.03 ~ 2022.10.19 |
| 인력 구성도(기여도) | 조석영(100%) |

## 주요 업무 및 상세 역할

### Swagger UI 활용
- Kakao Enterprise에서 제공하는 Docker Image들을 활용하여 CDC 구축
- Swagger UI에서 Metadata, Task, Step, Flow 생성
- Mapping, Compile 후 DML DDL 테스트

### Kafka 활용
- Kafka Topic, DB Connector를 활용하여 Source DB와 Sink DB 동기화
