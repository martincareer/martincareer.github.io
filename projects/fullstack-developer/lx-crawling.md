## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 작업 기간 | 2024.03 ~ 2024.10 |
| 인력 구성도(기여도) | 조석영(20%), 4명의 팀원 (80%) |
| 프로젝트 목적 | 사업 부서 별 Crawling 데이터 및 자동화 서비스 제공 |
| 프로젝트 내용 | 고객 친화적인 Crawling Node 기능 및 Data 제공 |
| 사용 언어 및 개발 환경 | Smart 9.1(Polymer), Docker, JS, Springboot(Java), MySQL, Python |
| 결과 | 고객 감사(우수)상 수상 |

---

## 주요 업무 및 상세 역할

### Full Stack 개발자(팀원)
- 요구사항 분석 및 WBS 설계

### Front-end & Back-end
- [개선] Crawling Node 오류 및 기능 개선
  - API를 활용한 자동 crawling
  - 여러 URL을 parameter로 받아 한 번에 crawling
  - Crawling 이후 발생하는 Alert handling

### Front-end
- 고객사 요청 URL 별 시나리오 개발
- 고객사 미팅 및 협의
