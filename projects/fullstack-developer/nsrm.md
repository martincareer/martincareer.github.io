## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 작업 기간 | 2024.04 ~ 2025.04 |
| 인력 구성도(기여도) | 조석영(20%), 4명의 팀원 (80%) |
| 프로젝트 목적 | AI를 활용한 공급망 관리 시스템 구축 |
| 프로젝트 내용 | 사업 별 공급망 현황 관련 Data Streaming 및 Dashboard 제공 |
| 사용 언어 및 개발 환경 | Smart 9.1(Polymer), Docker, JS, Springboot(Java), MySQL, Python |

## 주요 업무 및 상세 역할

### Full Stack 개발자(팀원)
- 요구사항 분석 및 WBS 설계
- Crawling 및 Data 적재
- DB Migration

### Front-end & Back-end
- [개발] POC Dashboard 개발 및 시연
- [개발] 엑셀 대용량 조회 및 다운로드 기능
- [개발] List, Pivot Table 개발
- [개선] Web Crawling Node 자동화

### Back-end
- [개발] QA 서버 환경 구성
- [개발] QA 플랫폼 환경 구성
- [개선] Parquet 및 csv 생성 로직 개선
- 고객사 미팅 및 협의
